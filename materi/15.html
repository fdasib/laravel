Page expired

  • Page Expired : ini terjadi karena laravel berusaha mengamankan halaman web kita ketika kita menggunakan post
  • buka didokumetasi
  • CRSF : cross-site request forgery
  • CSRF : sebuah teknik serangan terhadap website kita, dimana serangan ini memalsukan (forgery == memalsukan) request dari website lain
  • menjaga request agar selalu dikirimkan dari website kita dengan csrf_toker
  • jadi website kita akan mengenerate sebuah token didalam session dan akan dicocokkan dengan yang ada didalam sesionnya
  • cukup menambahkan @csrf(blade directive) akan menerjemahkan seperti input hidden
