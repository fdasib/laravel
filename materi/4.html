Blade Templating Engine

Templating Engine

  • Sebuah fitur atau tools untuk membantu kita dalam mengelola tampilan halaman web khususnya untuk sebuah framework
  • untuk laravel Templating Engine nya dinamakan blade

Penjelasan Blade

  • buka dokumentasi blade : https://laravel.com/docs/9.x/blade
  • blade itu adalah sebuah templating engine yang sudah terdapat didalam laravel
  • didalam nya kita masih bisa menggunakan syntax php didalam view kita dengan tambahan syntax blade
  • pada akhirnya akan dicompile kedalam syntax PHP biasa

Displaying Data dengan Blade

  • tidak usah menggunakan PHP echo
  • ini adalah syntax echo milih blade
  • fungsinya sama dengan PHP echo tapi ada kelebihannya
  • otomatis dibelakang layar sudah ditambahkan sebuah fungsi htmlspecialchars
  • yang sangat berguna untuk menghindari serangan XSS(Cross Side Scripting)
  • kalo perlu install extension bernama : Laravel Blade Spacer
  • buka file about.blade.php

  • untuk melihatnya ada dibagian storage/framework/views
  • ada banyak file, file ini otomatis dibuat laravelnya untuk menampilkan halaman view hasil compilenya
