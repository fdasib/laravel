Intro

Apa itu Laravel?

  • Sebuah Kerangka aplikasi web dengan syntax yang elegant dan juga expresive

Web Application Framework

  • Sebuah 'kerangka' yang didesain untuk mendukung pembangunan aplikasi web
  • Menyediakan standar / cara untuk membangun aplikasi web
  • Bertujuan untuk mengotomasi hal2 umum yang biasanya dilakukan saat membangun aplikasi web
  • Contohnya: database library, templating engine, session, management, authentication, security, dll
    • database library : untuk koneksi ke database, query ke database
    • templating engine : bagaimana cara kita agar dapat dengan mudah menampilkan data yang kita butuhkan ke dalam halaman webnya, bagaimana cara kita memisahkan yang tertentu dari halaman web

Pembuat laravel

  • Dibuat oleh Taylor Otwell
  • Juni 2011
  • Sebagai alternatif framework untuk CodeIgniter
  • Versi Terbaru 8

Filosofi

  • kita percaya kalo proses pengembangan aplikasi / perangkat lunak itu harus bisa dinikmati dan juga merupakan pengalaman creative yang bisa memberikan kepuasan / kebahagiaan

Tujuan Laravel

  • Membuat proses pengembangan itu bisa dinikmati oleh developernya tanpa harus mengorbankan fungsionalitas aplikasinya
  • ngoding harus bisa dinikmati, karena jika developernya happy, codingan yang dia buat juga pasti yang terbaik

Fitur utama pada laravel

  • MVC
    • Sebuah paradigma yang memisahkan antara tampilan, proses dan juga data
      • Model, View, Controller
    • Templating Engine
    • Artisan Console
      • Fitur yang kita bisa melakukan modifikasi / konfigurasi terhadap frameworknya hanya dengan mengetikkan baris2 perintah diconsolenya
      • diconsole yang namanya artisan
    • Eloquent ORM (Object Relasional mapping)
      • yang mempermudah kita, ketika kita nanti mau berinteraksi dengan database yang relasional, seperti MySQL
    • Authentication & Authorization
      • ketika mau bikin fitur login dan registrasi
    • Testing
    • Packaging system
    • Multiple File System
    • Task Scheduling
    • Websocket Programming

    Ekosistem Laravel

    • ketika kita mau menggunakan layanan apapun, menggunakan fitur apapun, laravel ini sudah punya banyak selain frameworknya itu sendiri
    • misalnya ketika kita ingin membuat sebuah platform yang serverless sudah ada product yang namanya Vapor
    • Mengelola server : Forge
    • Mengelola Local Docker Environment : Sail
